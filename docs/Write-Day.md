# Write-Day

## SYNOPSIS
Simply outputs the day

## SYNTAX

```
Write-Day [<CommonParameters>]
```

## DESCRIPTION
Literally just outputs today's day

## EXAMPLES

### Example 1
```powershell
PS C:\> Write-Day
```

Will output `Friday`.
